#To-Do App (C# Windows Forms)

# Proje Hakkında
Bu proje, C# Windows Forms kullanılarak geliştirilmiş basit bir To-Do uygulamasıdır.
Eğer uygulamaya ilk defa giriliyorsa kullanım kılavuzu açılmaktadır.
İlk defa girilmiyorsa uygulama kullanılmaya başlanabilir. Kullanıcı ihtiyaç duyarsa kullanım kılavuzunu tekrar açabilmektedir.
Kullanıcılar görev ekleyebilir; tamamlanan görevleri işaretleyebilir, silebilir ve güncelleyebilir.

---
## Özellikler
- Yeni görev ekleme
- Tarih ekleme (varsa)
- Görevleri işaretleme 
- Seçili görevi silme
- Seçili görevi ekleme
- Kullanma Kılavuzu
